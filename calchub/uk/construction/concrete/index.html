Часті запитання

Скільки цементу на 1 куб бетону М200?▼
Для М200 потрібно ~320–350 кг цементу на 1 м³. Це 6–7 мішків по 50 кг.
Яка товщина стяжки для підлоги?▼
Мінімум 30 мм, рекомендована — 50–70 мм. Для теплої підлоги — не менше 50 мм над трубами.
Яка марка бетону для фундаменту?▼
Для стрічкового фундаменту — М200, для плити — М250–М300. М100 тільки для підготовчого шару.
