Калькулятор бетону для фундаменту та стяжки онлайн

Онлайн калькулятор бетону допоможе точно розрахувати об'єм суміші та кількість матеріалів для будь-якої конструкції: фундаменту, монолітної плити, стяжки підлоги або доріжки. Введіть розміри та марку бетону — і отримайте готовий список матеріалів.

Калькулятор показує витрату цементу в кілограмах та мішках, піску і щебеню в кілограмах, а також необхідну кількість води. Підтримуються марки бетону від М100 до М300 — найпоширеніші у приватному будівництві України.

Часті запитання

Скільки цементу на 1 куб бетону М200?▼
Для бетону М200 потрібно приблизно 320–350 кг цементу на 1 м³ — це 6–7 мішків по 50 кг. Пропорція для М200: 1 частина цементу, 2,8 частини піску, 4,8 частини щебеню.
Яка товщина стяжки для підлоги?▼
Мінімальна товщина стяжки — 30 мм, рекомендована для житлових приміщень — 50–70 мм. Для теплої підлоги з водяними трубами — не менше 50 мм над трубами. Занадто тонка стяжка тріскається, занадто товста — збільшує навантаження на перекриття.
Яка марка бетону для фундаменту?▼
Для стрічкового фундаменту приватного будинку рекомендується М200, для монолітної плити — М250–М300. М100 використовується тільки для підготовчого шару (підбетонки). Для гаражу або господарської будівлі достатньо М150–М200.
Яке співвідношення цементу, піску і щебеню для бетону?▼
Класичне співвідношення для М200: цемент : пісок : щебінь = 1 : 2,8 : 4,8. Для М300 — 1 : 1,9 : 3,7. Водоцементне відношення — 0,5–0,6 (на 1 кг цементу 500–600 мл води). Менше води — міцніший бетон.
Скільки бетону потрібно на 1 м² стяжки 50 мм?▼
На 1 м² стяжки товщиною 50 мм потрібно 0,05 м³ бетону. Для кімнати 20 м² — 1 м³ суміші: приблизно 320 кг цементу, 700 кг піску та 1200 кг щебеню. Скористайтесь калькулятором для точного розрахунку ваших розмірів.
