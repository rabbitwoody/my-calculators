Часті запитання

Як розрахувати ідеальну вагу?▼
Найпопулярніша формула — Брока: для чоловіків ідеальна вага = зріст (см) – 100, для жінок = зріст (см) – 110. Результат коригується залежно від типу статури.
Яка різниця між формулами Брока та Девіна?▼
Формула Брока проста і популярна, але розроблена для людей зростом 155–185 см. Формула Девіна точніша для різного зросту і широко використовується в медицині.
Що таке тип статури?▼
Тип статури (соматотип) визначається шириною кісток. Дрібна (астенічна) — вузькі кістки, ідеальна вага менша. Середня (нормостенічна) — стандарт. Велика (гіперстенічна) — широкі кістки, вага більша.
