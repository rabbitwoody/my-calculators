Часті запитання

Що таке пропорція?▼
Пропорція — рівність двох відношень: a/b = c/d. Використовується для масштабування, рецептів, карт.
Як знайти четвертий пропорційний?▼
Якщо a/b = c/x, то x = b×c/a. Приклад: 3/4 = 9/x → x = 4×9/3 = 12.
Як розподілити суму за пропорцією 3:2?▼
1000 грн у пропорції 3:2: сума частин=5. Перша = 1000×3/5 = 600 грн, друга = 400 грн.
