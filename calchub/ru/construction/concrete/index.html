Частые вопросы

Сколько цемента на 1 куб бетона М200?▼
Для М200 нужно ~320–350 кг цемента на 1 м³. Это 6–7 мешков по 50 кг.
Какая толщина стяжки для пола?▼
Минимум 30 мм, рекомендуемая — 50–70 мм. Для тёплого пола — не менее 50 мм над трубами.
Какая марка бетона для фундамента?▼
Для ленточного фундамента — М200, для плиты — М250–М300. М100 только для подготовительного слоя.
