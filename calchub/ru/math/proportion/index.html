Частые вопросы

Что такое пропорция?▼
Пропорция — равенство двух отношений: a/b = c/d. Используется для масштабирования, рецептов, карт.
Как найти четвёртый пропорциональный?▼
Если a/b = c/x, то x = b×c/a. Пример: 3/4 = 9/x → x = 4×9/3 = 12.
Как распределить сумму по пропорции 3:2?▼
1000 грн в пропорции 3:2: сумма частей=5. Первая = 1000×3/5 = 600 грн, вторая = 400 грн.
